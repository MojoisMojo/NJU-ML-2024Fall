# Prob2 Code

这一大题我们设置的随机种子为14;

## 文件解释

[main.py](./main.py) 

[layers.py](layers.py) 完成感知机各个层地实现\
[params.py](./params.py) 参数\
[model.py](./model.py) 感知机的模型\
[dataloader.py](./dataloader.py) 数据输入处理\
[utils.py](utils.py) 辅助函数

## 运行

如需修改参数，在model.py 和 params.py 中修改

```js
python main.py 或 model.py
```
